## 📋 Cara Penggunaan

### 1. Buka Website
Buka link demo di atas atau hosting sendiri.

### 2. Kelola Produk
- Klik tombol **⚙️ Admin Panel** di pojok kanan atas
- Tambah produk baru via form
- Edit atau hapus produk yang ada

### 3. Dapatkan Link Affiliate
- Login ke [Shopee Affiliate](https://affiliate.shopee.co.id)
- Pilih produk → klik **"Dapatkan Link"**
- Paste link ke form admin

### 4. Backup Data
- Klik **📥 Export Data** untuk backup
- Simpan file JSON di komputer
- Klik **📤 Import Data** untuk restore

## 🚀 Cara Deploy

### GitHub Pages (Gratis)
1. Fork repository ini
2. Buka **Settings** → **Pages**
3. Pilih branch **main**, folder **root**
4. Website online dalam 2-5 menit!

### Netlify (Gratis)
1. Buka [netlify.com](https://netlify.com)
2. Drag & drop file `index.html`
3. Website online instant!

## 🎨 Customisasi

### Ganti Warna Tema
Edit bagian `:root` di CSS:
```css
:root {
  --shopee-orange: #ee4d2d;   /* Ganti warna utama */
  --shopee-red: #d0011b;       /* Ganti warna sekunder */
  --shopee-yellow: #f6a700;   /* Ganti warna aksen */
}
```

### Tambah Produk Default
Edit array `defaultProducts` di JavaScript:
```javascript
const defaultProducts = [
  {
    id: 1,
    name: "Nama Produk",
    link: "https://shopee.co.id/...",
    priceOriginal: 1000000,
    priceDiscount: 500000,
    discount: 50,
    rating: 4.9,
    icon: "📱",
    color: "gradient1",
    sold: "10K"
  }
];
```
